\# HW07：胸部X光肺炎检测



\## 环境与运行方式

\- 在 Kaggle Notebook 中运行 `train.ipynb`。

\- 需挂载数据集：`paultimothymooney/chest-xray-pneumonia`。

\- 所有依赖均为 Kaggle 默认环境，也可用 `requirements.txt` 安装。



\## 数据集获取

Kaggle 上直接搜索 “Chest X-Ray Images (Pneumonia)”并 Add Data，或通过 kagglehub 下载。



\## 最终测试集指标

| 指标 | 数值 |

|------|------|

| Accuracy | 0.8397 |

| Precision | 0.8033 |

| Recall | 0.9846 |

| F1 Score | 0.8848 |
